# Write Skill

Creates or updates a skill definition file.

## Summary
Writes content to a skill file, creating the skill directory if needed.

## Prompt
Write the provided content to a skill file. Create the directory if it doesn't exist.

## Arguments
- skillName: Name of the skill (becomes directory name)
- fileName: Name of the file to write (e.g., cskill.md, tskill.md)
- content: The content to write to the file
